# 📚 Kitap Atlası (Digital Story Map)

Kitap Atlası, Türkiye'nin edebiyat ve kültür haritasını görselleştiren, kullanıcı dostu ve interaktif bir masaüstü uygulamasıdır. Bu proje, okurların kitapları şehirlerle eşleştirmesine ve istatistiksel veriler üzerinden bir "okuma haritası" oluşturmasına olanak tanır.

## ✨ Özellikler

* **Interaktif Harita:** Türkiye haritası üzerinden şehir bazlı kitap takibi.
* **Gelişmiş UI/UX:** Dinamik dalga efektli açılış ekranı (Animated Splash Screen) ve karanlık tema desteği.
* **Veri Yönetimi:** JSON tabanlı veri saklama sistemi (cached_books.json).
* **İstatistik Paneli:** Okuma alışkanlıklarınızı analiz eden özel grafikler ve tablolar.
* **Arama ve Filtreleme:** Kitaplar arasında hızlı arama ve kategori bazlı filtreleme.

## 🛠️ Teknik Detaylar

Bu proje modern Python teknolojileri kullanılarak geliştirilmiştir:
* **Framework:** PySide6 (Qt for Python)
* **Programlama Dili:** Python 3.x
* **Veri Yapısı:** JSON & SQL entegrasyonu
* **Tasarım:** Özel QSS (Qt Style Sheets) temaları
